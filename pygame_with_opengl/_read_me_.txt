Чтобы игра работала, необходимо:
1. Создать в папке виртуальную среду и активировать её
2. Установить для виртуальной среды Pygame
3. Установить для виртуальной среды Numpy
4. Установить для виртуальной среды PyOpenGL
4. Запустить файл game.py

Все эти шаги описаны в видео: https://youtu.be/KYxITPlDDZI
